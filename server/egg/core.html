# 日志

# 日志分类

  • appLogger 应用相关日志

  • coreLogger 框架内核，插件日志

  • errorLogger 任何调用error输出的日志都会重定向到这里

  • agentLogger agent日志 agent进程执行任务插件会输出到这里
